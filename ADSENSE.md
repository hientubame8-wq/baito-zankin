# Google AdSense 申請ガイド（あと稼げるくん）

このサイトは AdSense 審査の準備が整っています。以下は **あなた自身の Google アカウントで行う手順**です（自動化できない部分）。

公開URL: https://hientubame8-wq.github.io/baito-zankin/

---

## ステップ1. AdSense アカウントを作成

1. https://adsense.google.com にアクセスし、Google アカウントでログイン
2. 「ご利用開始」→ サイトのURL `https://hientubame8-wq.github.io/baito-zankin/` を入力
3. 国（日本）・支払い通貨を設定し、利用規約に同意

## ステップ2. サイトに広告コードを設置

1. AdSense管理画面で発行される **「ca-pub-数字16桁」** をコピー
2. `index.html` の「Google AdSense ここから」コメント内の `<script>` 行を有効化:
   - 行頭・行末の `[` と `]` を削除（コメント解除）
   - `ca-pub-XXXXXXXXXXXXXXXX` を自分の番号に置換
3. `ads.txt` の `pub-XXXXXXXXXXXXXXXX` も同じ番号に置換
4. 変更を commit & push:
   ```bash
   git add index.html ads.txt
   git commit -m "AdSense発行者IDを設定"
   git push
   ```

## ステップ3. 審査を待つ

- AdSense管理画面で「審査をリクエスト」
- 結果は通常 **数日〜2週間**でメール通知
- 承認されると「自動広告」がページに自動表示されます（手動設定不要）

## 審査に通りやすくするためのチェックリスト

このサイトは以下を **すでに満たしています**:

- ✅ オリジナルで有用なコンテンツ（年収の壁ガイド・FAQ）
- ✅ プライバシーポリシー（Cookie・広告について明記）
- ✅ 免責事項
- ✅ 実用的なツール本体
- ✅ スマホ対応・明確なナビゲーション
- ✅ HTTPS / sitemap.xml / robots.txt / 構造化データ

## ⚠️ 審査で不利になりうる点と対策

| 懸念 | 対策 |
|---|---|
| `github.io` サブドメイン | **独自ドメイン**（年1,000〜1,500円）取得で通過率が上がる |
| 1ページ構成（情報が薄いと判断されやすい） | 「年収の壁」関連の記事ページを2〜3本追加すると安全 |
| 新規サイトでアクセスが少ない | SNS/noteで導線を作り、数週間運用してから申請 |

**おすすめ**: まず現状で申請 → もし却下されたら、独自ドメイン取得＋記事追加で再申請。
