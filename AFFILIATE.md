# アフィリエイト設定ガイド（あと稼げるくん）

このサイトには広告枠が**4か所**用意されています（ツールのガイドタブ＋記事3本）。
リンク未設定の枠は**自動的に非表示**になるので、設定するまで訪問者には見えません。

公開URL: https://hientubame8-wq.github.io/baito-zankin/

---

## ステップ1. アフィリエイトサービスに登録（無料）

github.io でも使える、審査がゆるめのものがおすすめです。

| サービス | 特徴 | 登録 |
|---|---|---|
| **もしもアフィリエイト** | 初心者向け・審査ゆるめ・Amazon/楽天もまとめて | https://af.moshimo.com/ |
| **A8.net** | 案件数No.1・サイト審査なしで登録可 | https://www.a8.net/ |

登録時にこのサイトのURLを登録します。

## ステップ2. 紹介する広告主と提携

このサイトのコンテンツと相性が良い案件：

**① バイト求人（枠: `baito_kyujin`）**
- マイナビバイト / バイトル / シフトワークス など
- A8.net・もしもで「バイト 求人」で検索して提携申請

**② 確定申告・会計ソフト（枠: `kakutei_shinkoku`）**
- freee / マネーフォワード クラウド確定申告 / やよいの白色申告 オンライン
- A8.net で「確定申告」「会計ソフト」で検索して提携申請

提携が承認されると、**広告主ごとの「リンクURL（あなた専用）」**が発行されます。

## ステップ3. リンクを貼り付ける

`affiliate.js` を開き、発行されたURLを貼るだけ：

```js
const AFFILIATE = {
  baito_kyujin:     'https://px.a8.net/svt/ejp?... あなたの求人リンク',
  kakutei_shinkoku: 'https://px.a8.net/svt/ejp?... あなたの確定申告ソフトのリンク',
};
```

> リンクは「テキストリンク」のURLを使ってください（バナーHTMLではなくURL部分のみ）。

## ステップ4. 反映

```bash
git add affiliate.js
git commit -m "アフィリエイトリンクを設定"
git push
```

数分後、設定した枠に「広告」付きのボタンが表示されます。

## ルール（守らないと規約違反・法令違反になります）

- **「広告」表示は消さないでください**（2023年〜のステマ規制で必須）
- 虚偽・誇大な表現はしない（景品表示法）
- 各サービスの規約（自己クリック禁止など）を守る

## 収益の目安

- アフィリエイトは「クリック」より「成約」で報酬が出るタイプが主流
- 確定申告ソフト無料登録: 1件 数百〜数千円
- バイト求人 応募/登録: 1件 数百〜千円台
- まずは検索流入（SEO）を増やすことが収益の前提
